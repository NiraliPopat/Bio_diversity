IndexFiles.java indexes the corpus by dividing file in passages and performing required processing.

SearchFiles.java is used to search and get results using various scoring models.

Query.java is used to create file for normal preprossed query.

Query_aq.java is used to create file for query expansion using automatic weights.

Query.java is used to create file for query expansion using manual weights.

Wiki_aspect_Extraction.java is used to get wikipedia extracted aspects.

Meta_aspect_Extraction.java is used to get Metamap extracted aspects.

Rerank.java is used to rerank passages and get results.

MetamapEx.java is to get UMLS concepts from text using Metamap
